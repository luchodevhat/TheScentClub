# 🌸 The Scent Club – Full Stack Web Application

## 📌 Project Overview

Perfume Store is a full stack e-commerce web application designed to sell perfumes online. The platform allows users to browse products, create accounts, log in, and place orders. It also includes an administrator panel to manage products, users, and orders.

This project was built to strengthen full stack development skills and simulate a real-world online store.

---

## 🚀 Features

### 👤 User Features

* Browse perfumes by category
* View product details
* Purchase products
* WhatsApp integration for order confirmation

## 🔐 Authentication

The system uses authentication to allow access to private routes. Users and administrators have different roles and permissions.

* Users: Can browse and purchase
* Admins: Can manage products and orders

---

## 💬 WhatsApp Integration

The platform allows customers to complete orders through WhatsApp. Once the user selects products, the system generates a message with the order details and sends it directly to the store’s WhatsApp.

## 📦 Installation

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/perfume-store.git
```

### 2️⃣ Navigate to the project

```
cd perfume-store
```

### 3️⃣ Install backend dependencies

```
npm install
```

### 4️⃣ Configure environment variables

Create a `.env` file and add:

```
PORT=3000
DB_HOST=localhost
DB_USER=your_user
DB_PASSWORD=your_password
DB_NAME=perfume_store
JWT_SECRET=your_secret
```

### 5️⃣ Run the backend server

```
npm run dev
```
